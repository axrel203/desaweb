# Web Desa

Project website untuk desa yang dibangun menggunakan Vite.

## Tech Stack
- Vite
- Vanilla JS / HTML / CSS
- AOS (Animate On Scroll)

## Cara Menjalankan Project

1. Install dependencies:
   ```bash
   npm install
   ```

2. Jalankan server development:
   ```bash
   npm run dev
   ```

3. Build untuk production:
   ```bash
   npm run build
   ```
